## Installation

Install yarn into your system

```bash
  npm install --global yarn
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/ratnakar5938/react-todo-list.git
```

Go to the project directory

```bash
  cd react-todo-list
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
